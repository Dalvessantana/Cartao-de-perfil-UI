# Cartao-de-perfil-UI
Projeto de cartão de perfil desenvolvido com HTML e CSS, com foco em design moderno e responsivo. Apresenta informações básicas do usuário, foto de perfil, botões de ação e ícones de redes sociais, seguindo uma interface limpa e intuitiva. Ideal para treinar front-end e criação de layouts estilizados.
